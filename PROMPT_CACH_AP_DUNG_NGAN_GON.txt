Viết hướng dẫn cách áp dụng và triển khai hệ thống gợi ý sản phẩm e-commerce với kiến trúc:

Frontend (React) → Backend (Node.js) → ML Models (Python/TensorFlow) → Database (MySQL)

BAO GỒM CÁC PHẦN:

1. CHUẨN BỊ:
   - Cài Node.js, Python 3.x, MySQL
   - Clone project và cài dependencies (npm install, pip install)
   - Tạo database và chạy migrations
   - Cấu hình .env (DB_HOST, DB_USER, DB_PASS, DB_NAME, PYTHON_BIN)

2. SETUP:
   - Backend: npm install → config database → start server
   - Python: pip install tensorflow pandas numpy → verify models (.h5 files)
   - Frontend: npm install → config API endpoint → start dev server

3. DỮ LIỆU:
   - Import products vào database
   - Tạo encoders (user_encoder, item_encoder) 
   - Migrate encoders vào DB (rec_user_encoder, rec_item_encoder)
   - Load context mappings

4. MODELS:
   - Verify 4 model files: encm.weights.h5, lncm.weights.h5, neumf.weights.h5, bmf.weights.h5
   - Verify config files: *_config.json
   - Test model loading

5. TRIỂN KHAI:
   - Start Backend API (port 3000)
   - Test Python invoker từ Node.js
   - Start Frontend (port 3001)
   - Test API endpoints: /api/recommendations/init, /api/recommendations/list

6. SỬ DỤNG:
   - API: GET /api/recommendations/init?limit=10 (khởi tạo)
   - API: GET /api/recommendations/list?limit=10 (lấy gợi ý)
   - Frontend: Gọi API và hiển thị sản phẩm
   - Track interactions: view, cart, purchase

7. TÍCH HỢP:
   - React component gọi API recommendations
   - Hiển thị danh sách sản phẩm gợi ý
   - Auto-track user interactions
   - Auto-refresh khi có interaction mới

8. MONITORING:
   - Check logs
   - Xem metrics trong model_runs table
   - Monitor MAP@10, Precision@10
